Сессии пользователя

GET /sessions
Описание: Получить список всех сессий пользователя (устройств, где выполнен вход).
Ответ: 
{
  "sessions": [
    {
      "id": 1,
      "user_agent": "Mozilla/5.0 ...",
      "ip_address": "192.168.1.1",
      "created_at": "2024-06-07 12:00:00",
      "last_active": "2024-06-07 12:30:00",
      "is_active": 1
    },
    ...
  ]
}
Поля:
  • id — идентификатор сессии
  • user_agent — браузер/устройство
  • ip_address — IP-адрес
  • created_at — когда был выполнен вход
  • last_active — последнее действие
  • is_active — активна ли сессия (1 — да, 0 — нет)
DELETE /sessions/{id}
Описание: Завершить (отозвать) конкретную сессию пользователя.
Ответ: { "success": true }
